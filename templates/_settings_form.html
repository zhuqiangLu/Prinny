Connecting Zotero

Prinny only reads your library — it tries Zotero's local API first, then falls back to the SQLite file. Set up either one:

  1. Recommended — local API (keep Zotero open): in Zotero open Settings → Advanced → Miscellaneous and tick “Allow other applications on this computer to communicate with Zotero.” It exposes http://localhost:23119/api/ (the “HTTP API base” below). Restart Zotero if it was already running.
  2. Fallback — SQLite (Zotero must be closed): point the path below at ~/Zotero/zotero.sqlite. Zotero locks this file while it's open, so quit Zotero to use this mode — an open Zotero is the usual cause of “unable to open database file.”

Preferred when reachable; SQLite is the fallback.
